JavaEE - Exception Binder using Interceptor



Interceptor Binding is used to bind the thrown Exception and Json Response is send to Client dynamically.

Request method: GET method
Response format: JSON format

API for Custom Exception: localhost:8080/exceptionInterceptor-web/api/exception/custom
API for Runtime Exception: localhost:8080/exceptionInterceptor-web/api/exception/runtime
API for File Already Exist Exception: localhost:8080/exceptionInterceptor-web/api/exception/fileAlreadyExist
